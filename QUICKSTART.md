# 快速启动指南

## Windows 用户（推荐）

1. 双击运行 `start.bat`
2. 等待依赖安装完成（首次运行需要几分钟）
3. 浏览器自动打开或手动访问 http://localhost:5173

## 手动启动

### 步骤 1: 安装依赖

```bash
# 安装后端依赖
npm install

# 安装前端依赖
cd client
npm install
cd ..
```

### 步骤 2: 启动服务

```bash
npm run dev
```

### 步骤 3: 访问应用

打开浏览器访问：http://localhost:5173

## 使用流程

1. **创建连接**
   - 点击左侧"新建连接"按钮
   - 选择数据库类型（MySQL/PostgreSQL/SQLite/MongoDB）
   - 填写连接信息
   - 点击"测试连接"验证
   - 点击"保存"

2. **连接数据库**
   - 在连接列表中点击"连接"按钮
   - 输入密码
   - 点击"连接"

3. **浏览数据库**
   - 在左侧树形结构中展开数据库
   - 点击表名查看表结构和数据

4. **执行SQL查询**
   - 点击顶部"SQL查询"标签
   - 在编辑器中输入SQL语句
   - 选择数据库（可选）
   - 点击"执行"按钮

## 常见问题

### 端口被占用

如果5000或5173端口被占用，可以：

1. 修改后端端口：编辑 `.env` 文件，设置 `PORT=其他端口`
2. 修改前端端口：编辑 `client/vite.config.js` 中的 `server.port`

### 连接失败

- 检查数据库服务是否运行
- 检查连接信息是否正确
- 检查防火墙设置
- 对于远程数据库，确保允许远程连接

### 依赖安装失败

- 确保已安装 Node.js (版本 >= 16)
- 尝试清除缓存：`npm cache clean --force`
- 删除 `node_modules` 和 `package-lock.json` 后重新安装
